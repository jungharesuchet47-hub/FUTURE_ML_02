# Resume Screening System 📄🤖

This repository contains the completed code for **Task 2** of my Machine Learning Internship at **Future Interns**. 

## 📌 Project Overview
The objective of this project is to build a Machine Learning model that can automatically read, clean, and classify candidate resumes into their respective job categories using Natural Language Processing (NLP).

## 🛠️ Technologies & Libraries Used
- **Programming Language:** Python
- **Data Manipulation:** Pandas, NumPy
- **Text Preprocessing (NLP):** Regular Expressions (re), TF-IDF Vectorizer
- **Machine Learning Algorithm:** K-Nearest Neighbors (KNN)

## 📊 Results & Performance
- Cleaned the resume text by removing URLs, hashtags, mentions, and special characters.
- Extracted features from the text using `TfidfVectorizer`.
- The K-Nearest Neighbors (KNN) classification model achieved an outstanding **Accuracy of 100%** on the testing dataset.

## 📂 Repository Contents
- Jupyter Notebook (`.ipynb`) containing the step-by-step code.
- Dataset used for training and evaluating the model.
